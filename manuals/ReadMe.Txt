This subdirectory contains the relevant Delta Tau manuals in .pdf
format. 
You need Acrobat Reader (http://www.adobe.com/products/acrobat) in
order to read or print them.

Oberon manuals can be found in applic/docu.arc which should be
installed in the system.

The CNC manuals can be found in nc/nc.arc which should also be
installed in the system. Look for 'NC*.Text' in the Documents.Panel
or open System Help, the files are listed at the top of that file.
